## 📱 About the App

**My Overtime** is a free Android app designed for factory workers, supervisors, and small business owners in Bangladesh to easily calculate monthly salaries, overtime pay, night bills, and attendance bonuses — all offline, no internet required.

> 🇧🇩 বাংলা ও English — দুই ভাষায় ব্যবহার করুন।

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 👷 **Worker Management** | কর্মী যোগ, এডিট ও ডিলিট করুন |
| 💰 **Salary Calculation** | গ্রস বেতন, কর্তন, বোনাস সহ সম্পূর্ণ হিসাব |
| ⏱ **Overtime Tracking** | প্রতিদিনের OT ঘন্টা এন্ট্রি করুন |
| 🌙 **Night Bill Entry** | রাতের শিফটের বিল আলাদাভাবে ট্র্যাক করুন |
| 📊 **Monthly Reports** | মাসভিত্তিক রিপোর্ট ও রেকর্ড |
| ✅ **Paid/Unpaid Tracking** | কে বেতন পেয়েছে সহজে মার্ক করুন |
| 🌐 **Bilingual** | বাংলা ও English সাপোর্ট |
| 📴 **100% Offline** | কোনো internet লাগবে না |

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Expo CLI
- Android device or emulator

### Installation

```bash
# Clone the repository
git clone https://github.com/rsakashh/my-overtime.git

# Navigate to project folder
cd my-overtime

# Install dependencies
npm install

# Start the app
npx expo start
```
